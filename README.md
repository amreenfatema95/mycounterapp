1. INTRODUCTION
-----------------
A basic Android Application that has been coded in Java using Android Studio as the IDE. This application shows use of some widgets and associates an action listener to these widgets.
Basically the application contains a seekBar and two buttons that can be used to increase or decrease the movement of values on the seeker.In addition the seeker can be used on its own to increment or decrement the displayed Values.

2. External Libraries:
-----------------------
None

3. USAGE
---------

The application can used by running any emulator like Genymotion.
